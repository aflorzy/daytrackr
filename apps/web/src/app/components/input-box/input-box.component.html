Input data in the text area below. Each day should be separated by a newline (\n) and should begin with the day's name (Sunday, Monday... etc.) followed by '- ' and a list of the day's events (comma-separated).

eg. 'Saturday- Breakfast, work, Minecraft, lunch'

Enter the date of the first day in 'YYYY/MM/DD' format. It is validated by ensuring the day 'Saturday' corresponds to the name of the actual date '2023/08/05'. If the date is invalid, the 'Parse' button will be disabled. Each subsequent day listed is assumed to be the next day in the sequence.
