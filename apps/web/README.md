# DaytrackrParser

## Current Behavior

Interface for entering, editing, and viewing days and the events that took place.

Enter raw text
Parse text and assign a date depending on the weekday. Events are separated by commas
Combine events using 'join' button between events. This joins the text with a comma
Click 'save' to persist edited data to localStorage
View saved days by clicking through the calendar


## Desired Features

Add event to a day
Move event position in a day
Assign tags to events (ex. 'Performed with', location (City, State))


## Known Issues

Clicking 'Edit' on a day, reordering events, and clicking 'Cancel' does not reset the 'Edit' page event order

Edit pen appears on Home page in Day list
